## Critical Edition Plan

This is the source text for the OpenKuyper critical edition project.

### Edition Comparison

| Edition | Year | Language | Notes |
|---------|------|----------|-------|
| 1847 Luchtmans | 1847 | Dutch | First edition — **base text** |
| Van Dyke translation | 1989 | English | Wedge Publishing — with extensive commentary |
| Lexham Classics | 2018 | English | Van Dyke translation, streamlined |

### Apparatus Structure

The critical edition will include:
- **Textual apparatus:** Variants between 1847 edition and any later Dutch editions
- **Translation apparatus:** Notes on translation decisions, comparing with Van Dyke
- **Historical apparatus:** Contextual notes on persons, events, and references
- **Theological apparatus:** Cross-references to Scripture, confessions, and theological sources

### Workflow

1. **Base text:** DBNL XML (diplomatic transcription of 1847 edition)
2. **AI translation:** Chapter-by-chapter Dutch → English using LLM
3. **Human review:** Compare against Van Dyke where available
4. **Apparatus:** Add critical notes, cross-references, and annotations
5. **Output:** Parallel Dutch/English edition with apparatus
